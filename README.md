# JIglesias - renovação de site

Proposta estática para renovação do site JIglesias, baseada no conteúdo público atual de `jiglesias.com`.

## Conteúdo

- `index.html` - página principal da proposta.
- `styles.css` - layout, responsividade e direção visual.
- `script.js` - menu mobile e filtros de viaturas.

## Ver localmente

```bash
python3 -m http.server 8797
```

Depois abrir `http://127.0.0.1:8797/`.

## Publicação

O projeto pode ser publicado diretamente via GitHub Pages a partir da branch `main`.
